# Solution sketches

Most exercises in the book are open-ended by design. They ask you to collect
data, make defensible choices, and write up what you found, and there is rarely
one correct answer. Worked solutions would defeat the point.

What this folder offers instead is a starting point for the coding tasks: a
short sketch showing one reasonable way in, built on the synthetic data, so you
can check that your environment works and then take the analysis further. These
are scaffolds, not model answers. The judgement the book asks for is yours to
supply.

Instructors who would like a fuller answer guide and an assessment question
bank can contact the author through the details in the main README.
