# Chapter 11 exercises

## Agent-based modelling and simulation

- Coding task. Implement a minimal routine-activity model on a grid, in NetLogo or in Python with Mesa: offenders and targets move, an offence occurs when they meet without a guardian present. Show that spatial concentration of offences emerges from movement rules alone, with no hot spot written into the code.
- Coding task. For the model in Exercise 1, vary guardian density across five levels and run one hundred seeds at each. Plot the distribution of offence counts at each level, and report your results as distributions rather than as single numbers.
- Write a complete ODD description of your model, then exchange descriptions with a peer and attempt to rebuild each other’s model from the document alone. Report where the descriptions failed.
- Choose a published agent-based model in criminology and assess its validation: what empirical patterns was it tested against, were those patterns independent of the calibration targets, and what would falsify the model?

---

Coding tasks above can be started from the matching chapter script (`code/python/ch11_*.py` or `code/r/ch11_*.R`) and the synthetic data in `data/synthetic/`. Where an exercise asks for real data, the data directory README lists open sources you can substitute.
