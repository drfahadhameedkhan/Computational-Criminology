# Chapter 7 exercises

## Statistical foundations

- Coding task. Simulate crime counts under a Poisson process and under a negative binomial process with the same mean. Fit a Poisson regression to both, and report what overdispersion does to the standard errors and to the rate of false positive findings across repeated simulations.
- Coding task. Using incident data aggregated to small areas nested in larger districts, fit a single-level model and a multilevel model of counts on the same predictors. Compare the estimates and intervals, and explain in writing what partial pooling did and why.
- Take one research question and one dataset, and produce three analyses of it that a reasonable researcher could defend: different outcome definitions, covariate sets, or time windows. Report all three results side by side and reflect on what the spread implies about any single published estimate.
- Find a published regression table in a criminology journal. For each reported coefficient, state whether the surrounding text treats it as description or as cause, and whether the design entitles the authors to the stronger reading.

---

Coding tasks above can be started from the matching chapter script (`code/python/ch07_*.py` or `code/r/ch07_*.R`) and the synthetic data in `data/synthetic/`. Where an exercise asks for real data, the data directory README lists open sources you can substitute.
