# Contributing

This repository accompanies a book, so its first duty is to match the text and
to keep every example runnable. Contributions that help with either are
welcome.

The most useful contributions are these. Report a script that fails to run, or
a result that no longer matches the book, by opening an issue with the chapter,
the command you ran, and the full error. Improve a comment or a variable name
where it would help a newcomer, without changing what the book's code does.
Add an R equivalent or a worked solution scaffold where one is missing.

A few conventions keep the repository coherent. Keep the book's teaching code
unchanged inside each script, and mark any setup lines you add around it, as
the existing scripts do. Drive anything random from a fixed seed, so results
reproduce. Run the test commands in the README before opening a pull request,
and make sure the synthetic data still rebuilds and the figures still generate.
