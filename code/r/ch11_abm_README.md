# Chapter 11 - Agent-based modelling

The book presents the agent-based model in Python only, because the simulation
toolchain (Mesa, or plain NumPy) is the natural home for this method. There is
no paired R snippet in the text.

For an R reader who wants to follow along, the same routine-activity logic can
be written with base R using matrices and a movement loop, or with the
`simecol` package. The Python reference implementation is in
`code/python/ch11_abm.py`.
